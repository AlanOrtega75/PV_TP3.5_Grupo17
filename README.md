Plataforma de gestión de proyectos educativos hecha para la materia PV (Programación Visual).

El proyecto comenzó con HTML/CSS y durante el TP3 lo fuimos pasando a React de a poco. En cada
parte sumamos algo nuevo: componentes y manejo de estado con hooks, comunicación entre
componentes con props, efectos con useEffect, navegación entre vistas con React Router y el
diseño de la interfaz con Material UI. En esta parte agregamos estado global con Context API
para manejar los datos del usuario y compartirlos en toda la aplicación.

> El desarrollo del proyecto vive en la rama `develop`.
